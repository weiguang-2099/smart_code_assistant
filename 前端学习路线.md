## 阶段一：前端基础（2周）

### 1.1 HTML5 + CSS3 基础

**核心知识点**：
- 语义化标签（header, main, section, article）
- Flexbox 弹性布局
- CSS Grid 网格布局
- 响应式设计与媒体查询
- CSS 变量与主题切换

**学习资源**：
- MDN Web Docs（官方文档）
- CSS Tricks（Flexbox/Grid 可视化指南）

**与项目关联**：
- 代码编辑器布局（左侧文件树 + 中间编辑区 + 右侧预览）
- 响应式适配不同屏幕

---

### 1.2 JavaScript 核心

**核心知识点**：
- ES6+ 语法（let/const、箭头函数、解构赋值、展开运算符）
- 模块化（import/export）
- Promise 与 async/await
- 数组高阶方法（map、filter、reduce）
- 事件循环与异步编程

**与项目关联**：
- API 调用（获取 AI 生成结果）
- 异步状态管理
- 事件处理（编辑器操作、用户交互）

---

### 1.3 TypeScript 基础

**核心知识点**：
- 基础类型与类型注解
- 接口（Interface）与类型别名（Type）
- 泛型基础
- 类型守卫与类型断言
- 模块与命名空间

**与项目关联**：
- 定义 API 请求/响应类型
- 组件 Props 类型定义
- 代码编辑器事件类型

---

## 阶段二：React 核心（3周）

### 2.1 React 基础

**核心知识点**：
- JSX 语法与组件化思想
- 函数组件 vs 类组件（推荐函数组件）
- Props 传递与 Children
- 条件渲染与列表渲染
- 事件处理机制

**学习资源**：
- React 官方文档（新版 react.dev）
- 实践：搭建简单的待办事项应用

---

### 2.2 React Hooks

**核心知识点**：
- useState：状态管理
- useEffect：副作用处理
- useRef：DOM 引用与持久化值
- useMemo / useCallback：性能优化
- useContext：跨组件通信
- 自定义 Hooks 封装

**与项目关联**：
- 管理代码编辑器状态
- 处理 WebSocket 连接生命周期
- 封装 AI 请求 Hook

---

### 2.3 React 路由

**核心知识点**：
- React Router v6 基础
- 路由配置与嵌套路由
- 路由参数与查询参数
- 路由守卫（认证保护）
- 动态加载与代码分割

**与项目关联**：
- 页面结构：首页 / 编辑器 / 项目管理 / 设置
- 用户认证后的路由保护

---

### 2.4 状态管理

**选型建议**：Zustand（轻量）或 Redux Toolkit（复杂场景）

**核心知识点**：
- 全局状态 vs 本地状态的取舍
- Zustand 基础用法
- Redux Toolkit：createSlice、createAsyncThunk
- 状态持久化

**与项目关联**：
- 用户认证状态
- 当前项目/文件状态
- AI 任务队列状态
- 实时协作用户状态

---

## 阶段三：项目技术栈（3周）

### 3.1 Monaco Editor 代码编辑器

**核心知识点**：
- @monaco-editor/react 集成
- 编辑器配置（主题、语言、字体）
- 语法高亮与代码补全
- 编辑器事件监听（onChange、onMount）
- 多文件编辑与 Tab 管理
- Diff 编辑器（代码对比）
- 自定义语言与主题

**学习资源**：
- Monaco Editor 官方文档
- @monaco-editor/react GitHub 仓库

**与项目关联**：
- 核心代码编辑功能
- AI 生成代码的展示与编辑
- 代码审查的 Diff 展示

---

### 3.2 SignalR 实时通信

**核心知识点**：
- @microsoft/signalr 客户端使用
- Hub 连接建立与重连策略
- 事件订阅与发布
- 流式数据传输（Streaming）
- 连接状态管理

**与项目关联**：
- AI 代码生成的流式返回（打字机效果）
- 多人实时协作编辑
- 代码审查实时反馈

---

### 3.3 HTTP 请求与 API 集成

**核心知识点**：
- Axios / Fetch 封装
- 请求拦截器（Token 注入）
- 响应拦截器（错误处理）
- React Query / SWR（数据请求管理）
- API 状态管理（loading、error、data）

**与项目关联**：
- 调用 .NET 后端 API（认证、项目管理）
- 调用 FastAPI AI 服务（代码生成、审查）

---

### 3.4 UI 组件库与样式

**推荐方案**：Tailwind CSS + Headless UI / Radix UI

**核心知识点**：
- Tailwind CSS 原子化样式
- 响应式前缀（sm、md、lg）
- 暗色模式支持
- Headless UI 组件（Modal、Menu、Tab）
- 组件库封装思想

**与项目关联**：
- 统一的设计风格
- 代码编辑器面板布局
- 弹窗、菜单、Tab 等交互组件

## 阶段四：项目实战（4周）

### 4.1 项目基础架构

**目录结构建议**：
```
src/
├── components/          # 通用组件
│   ├── Editor/          # 代码编辑器相关
│   ├── Layout/          # 布局组件
│   └── UI/              # 基础 UI 组件
├── features/            # 功能模块
│   ├── auth/            # 认证模块
│   ├── project/         # 项目管理
│   ├── code-gen/        # 代码生成
│   ├── code-review/     # 代码审查
│   └── collaboration/   # 实时协作
├── hooks/               # 自定义 Hooks
├── services/            # API 服务封装
├── stores/              # 状态管理
├── types/               # TypeScript 类型定义
└── utils/               # 工具函数
```

---

### 4.2 核心功能模块

#### 模块一：用户认证
- 登录/注册页面
- JWT Token 管理
- 路由权限控制
- 用户信息展示

#### 模块二：项目管理
- 项目列表与创建
- 文件树组件
- 项目设置面板

#### 模块三：代码编辑器
- Monaco Editor 集成
- 多 Tab 文件编辑

- 快捷键支持

#### 模块四：AI 代码生成
- 需求输入面板
- 流式代码生成展示
- Agent 执行步骤可视化
- 生成结果的应用与编辑

#### 模块五：代码审查
- 代码提交审查
- 审查结果展示（评分、问题列表）
- Diff 对比视图
- 问题定位与跳转

#### 模块六：实时协作
- 多人在线状态
- 光标位置同步
- 代码变更实时同步
- 协作聊天面板

## 学习建议

### 学习节奏
1. **理论 + 实践结合**：每学完一个知识点立即动手练习
2. **项目驱动学习**：以实际需求为目标，边学边做
3. **不求完美**：先跑通再优化，避免过度设计

### 常见误区
- 不要一开始就追求最佳实践
- 不要同时学习太多框架
- 不要跳过基础直接学框架
